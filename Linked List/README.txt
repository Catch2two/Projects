getSize() Get the size of the list.
getFirstNode() Select first node
getLastNode() Select last node
append() Add a new node with a value to the end of the list.
prepend() Add a new node with a value to the begining of the list.
getNodeIndexAt() Get node from Index(Specific location on list).
getNodeValueAt() Find the value of a node
removeNodeFromEnd() Remove last object in list
clear() Clear list.
printList() Show/Print list.
